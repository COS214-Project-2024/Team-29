\doxysubsection{Detailed Description}
Specialised concrete Landmark\+Buidling class that represents a \mbox{\hyperlink{classMonument}{Monument}}. 

The \mbox{\hyperlink{classMonument}{Monument}} class is a specialised type of \mbox{\hyperlink{classLandmarkBuilding}{Landmark\+Building}} that includes the ability to accept visitor objects for extended operations.
