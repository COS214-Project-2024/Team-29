\doxysubsection{Detailed Description}
Represents an bus with specified cost, capacity, and transport type. 

The \mbox{\hyperlink{classBus}{Bus}} class is a specialised mode of transport within a city, providing an implementation specific to buses, including attributes for cost, type, and capacity.
