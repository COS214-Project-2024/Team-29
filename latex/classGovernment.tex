\doxysubsection{Detailed Description}
Manages the policies that can be implemented within a city to impact citizen satisfaction and budget. 

The \mbox{\hyperlink{classGovernment}{Government}} class handles a collection of policies and provides functionality to implement them, which reduces city\textquotesingle{}s budget and affects citizen satisfaction.
