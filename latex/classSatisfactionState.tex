\doxysubsection{Detailed Description}
Abstract class representing a satisfaction state with operations to adjust the city\textquotesingle{}s satisfaction level. 

The \mbox{\hyperlink{classSatisfactionState}{Satisfaction\+State}} class serves as a base for different satisfaction levels\+:~\newline
 -\/\mbox{\hyperlink{classVeryUnsatisfied}{Very\+Unsatisfied}}~\newline
 -\/\mbox{\hyperlink{classUnsatisfied}{Unsatisfied}}~\newline
 -\/\mbox{\hyperlink{classNeutral}{Neutral}}~\newline
 -\/\mbox{\hyperlink{classSatisfied}{Satisfied}}~\newline
 -\/\mbox{\hyperlink{classVerySatisfied}{Very\+Satisfied}}~\newline
 providing an interface for retrieving and modifying the satisfaction state of a city.
