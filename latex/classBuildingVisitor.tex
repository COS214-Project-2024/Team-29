\doxysubsection{Detailed Description}
Abstract base class defining visitor operations for different types of buildings. 

The \mbox{\hyperlink{classBuildingVisitor}{Building\+Visitor}} class provides an interface for visiting various types of buildings, allowing different actions to be performed on each type. Derived classes implement specific visit behaviors.
