\doxysubsection{Detailed Description}
Represents an train with specified cost, capacity, and transport type. 

The \mbox{\hyperlink{classTrain}{Train}} class is a specialised mode of transport within a city, providing an implementation specific to airports, including attributes for cost, type, and capacity.
