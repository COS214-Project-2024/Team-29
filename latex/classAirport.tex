\doxysubsection{Detailed Description}
Represents an airport with specified cost, capacity, and transport type. 

The \mbox{\hyperlink{classAirport}{Airport}} class is a specialised mode of transport within a city, providing an implementation specific to airports, including attributes for cost, type, and capacity.
