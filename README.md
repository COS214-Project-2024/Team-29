## Compiling Code:
1. Clone repo to directory of choice (e.g /projects)
2. cd TheCityKings    (Directory of cloned repo)
3. mkdir build        (If not created already)
4. rm -rf build/*     (Fresh build, skip if new)
5. cd build
6. cmake ..
7. make
8. cd src
9. ./TheCityKings
