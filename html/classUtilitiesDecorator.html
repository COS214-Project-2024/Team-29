Detailed Description

The base decorator class for the various utility systems.

The UtilitiesDecorator class provides basic attributes and methods for managing the level and capacity of utility systems. It offers functionality for upgrading levels, checking capacity, and determining efficiency based on demand.

bool UtilitiesDecorator::isEfficient ( int  demand)

Checks if the utility is efficient given a demand.

Determines if the utility’s capacity at the current level can meet the demand. If demand is greater than capacity * 2^level, it returns false; otherwise, true.

Parameters
demandThe demand to check efficiency against.
Returns
True if the utility is efficient; false otherwise.
