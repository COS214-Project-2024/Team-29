Detailed Description

Represents an landmark building with specified tax income, construction cost, and satisfaction value.

The LandmarkBuilding class acts as an abstarct base class for the concrete classes: -CulturalCentre -Monument -Park<br>
provides the var of satisfactionValue specific to Landmark types of buidlings
