Detailed Description

Represents a commercial building with specified tax income, construction cost, and employee capacity.

The CommercialBuilding class acts as an abstarct base class for the concrete classes: -Mall -Office -Shop<br>
provides the var of employeeCapacity specific to Commercial types of buidlings
