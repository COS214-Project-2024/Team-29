Detailed Description

Abstract class representing a satisfaction state with operations to adjust the city's satisfaction level.

The SatisfactionState class serves as a base for different satisfaction levels:
-VeryUnsatisfied
-Unsatisfied
-Neutral
-Satisfied
-VerySatisfied
providing an interface for retrieving and modifying the satisfaction state of a city.
