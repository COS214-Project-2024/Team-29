Detailed Description

The abstract class for the factory method for creating buildings.

The BuildingCreator class provides the abstract template and method for the concrete factory classes (CommercialBuildingCreator, IndustrialBuildingCreator, LandmarkBuildingCreator, ResidentialBuildingCreator)
